**NULL** = **N**j**U**st cs **L**ost **L**essons.

## 本课程的目标

NULL 的目标不是让同学们看完之后就立刻掌握所有工具的使用，它的目标仅仅是想让同学们树立学习使用合适工具的意识。

如果未来某一天你遇到了一个需要解决的计算机问题，这时你想到：哦！我之前我在这门课了解过，某某工具可以帮助我解决这个问题！

这就是 NULL 期待达到的效果。
